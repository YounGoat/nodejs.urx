
NAME
    urx - Examine URLs in a markdown file.

SYNOPSIS
    urx [--file] <filename> [--longurl <style>] [--verbose]
    urx --help

OPTIONS
    --help
    Display this help info.

    --file <filename>
    Path of markdown file.    

    --longurl <style>
    What to do when URL is too long.
    The style is one of
        "keephead"
        "keeptail"
        "keepbody"
        "keepends"
        "cuthead"
        "cuttail"
        "cutends"

    --verbose, -v
    Display details of response or exception.
